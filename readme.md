# Aekino

![](public/assets/aekino.jpg)

Prototype of WebGL render engine  
[namide.github.io/aekino](https://namide.github.io/aekino)

## Commands

```bash
# Install
npm install

# Build samples
npm run build

# Run development server for samples
npm run dev-basics
npm run dev-load
npm run dev-relationship

# Convert FBX file to json
npm run fbx
```
